# Financially Correct Systems

A multi-module Java 17 and Spring Boot project demonstrating core patterns and implementations for building robust, financially correct systems. 

## Modules

* **`double-entry-ledger`**: Implementation of a double-entry accounting ledger to guarantee financial consistency.
* **`idempotent-payment-processor`**: A payment processing engine that strictly adheres to idempotency principles to prevent duplicate charges.
* **`reconciliation-engine`**: A system to compare and reconcile financial records across different domains or external partners.

## Technologies Used
* Java 17
* Spring Boot 3.2.3
* Maven
* Docker Compose
